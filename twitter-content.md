# 🐦 Twitter/X Content Prompt Pack

*9 prompts for high-engagement X content*

---

## 1. Viral Hook Tweet

**Purpose:** Capture attention in the first 2 seconds of a scroll

```
You are a viral Twitter writer. Write a HOOK TWEET that stops the scroll.

Write about: [TOPIC]

Requirements:
- Maximum 280 characters
- Start with a power word: "This", "Here's", "Actually", "Nobody", "Finally", "Stop"
- Create curiosity gap or counterintuitive statement
- Use concrete numbers or specificity when possible
- End with implied "keep reading"
- No threads, single tweet only

Output ONLY the tweet. No explanation.
```

---

## 2. Thread Starter

**Purpose:** Dive deep on a topic to build authority and reach

```
You are a thought leader on Twitter. Write a THREAD STARTER about:

[TOPIC]

Requirements:
- Hook (1 tweet): Bold claim or "1/"
- Value (3-7 tweets): Each tweet adds new insight
- Each tweet ends naturally to bait a follow
- Use specific examples, frameworks, or data points
- End with CTA: follow, quote tweet, or question
- Total thread: 4-10 tweets

Format each tweet on its own line, numbered 1-10.
```

---

## 3. Opinion/Polarizing Take

**Purpose:** Spark debate, generate engagement, position yourself

```
Write a CONTROVERSIAL OPINION TWEET about:

[TOPIC]

Requirements:
- 1-2 sentences max
- Specific enough to be actionable, vague enough to be debatable
- Use "I believe" or "Hot take:" or "Unpopular opinion:"
- Pick a side. Don't hedge.
- Make people nod or makes them angry—aim for both
- Avoid safe, agreeable statements

Output ONLY the tweet.
```

---

## 4. Quote Retweet with Commentary

**Purpose:** Join a conversation, add value, hijack attention

```
Original post to quote:
[PASTE POST]

Write a QUOTE RETWEET that:
- Adds a fresh perspective or angle
- Doesn't just agree—builds, challenges, or extends
- Is shorter than the original (or equal if it's tight)
- Includes your voice, not just "this"
- Has its own standalone value

Output only the commentary part (the text that will appear in your quote).
```

---

## 5. Announcement/Product Launch

**Purpose:** Generate buzz and conversions for something new

```
Write a PRODUCT LAUNCH TWEET for:

Product: [NAME]
What it does: [DESCRIPTION]
Target audience: [WHO]
Launch incentive: [OFFER/DATE]

Requirements:
- Lead with the outcome, not the feature
- Add social proof or credibility
- Create urgency or exclusivity
- Include a clear CTA
- Keep under 280 chars
- Make it feel like a movement, not a brochure

Output the tweet.
```

---

## 6. Engagement Question

**Purpose:** Start conversations, boost reply rate, learn from audience

```
Write an ENGAGEMENT QUESTION about:

[TOPIC or AREA]

Requirements:
- Ask something people can relate to
- Binary choices work well: "A or B?"
- "Who else" or "Drop a" patterns drive comments
- Make it low-effort to answer (one word works)
- Avoid yes/no unless it's provocative
- Add context only if needed

Output the question tweet.
```

---

## 7. Behind-the-Scenes

**Purpose:** Humanize your brand, build trust, content flywheel

```
Write a BEHIND-THE-SCENES TWEET about:

[WHAT YOU'RE SHARING - project, process, decision, failure, etc.]

Requirements:
- Authenticity over polish
- Show the messy reality
- Include a personal insight or lesson
- Make it relatable to your audience
- Optional: include "POV:" or "Storytime:"
- Make people feel included in your journey

Output the tweet.
```

---

## 8. Thread Recap

**Purpose:** Repurpose thread content, reach new audience, give summary

```
Write a THREAD RECAP from this thread:

[PASTE THREAD OR KEY POINTS]

Requirements:
- Summarize the core insight in 1-2 sentences
- Use numbered key takeaways (3-5 points)
- End with "Save this" or "Share with someone who needs this"
- Link back to the original thread if applicable
- Make it skimmable with line breaks

Output the recap tweet(s).
```

---

## 9. Reply to Trending Topic

**Purpose:** Insert yourself into existing conversation, build visibility

```
Trending topic: [TOPIC/HASHTAG]
Your angle/take: [YOUR UNIQUE PERSPECTIVE]

Write a REPLY that:
- Adds genuine value to the conversation
- Has a distinct opinion or insight
- Is brief but memorable
- Doesn't just agree—adds fuel
- Can stand alone if quote-tweeted

Output your reply.
```
